PAGES & ROUTES
================================================================================

Public Routes:
/ → Redirects to /login (if not authenticated) or /search (if authenticated)
/login → Login page with Google OAuth

Protected Routes (require authentication):
/search → Student search page
/student/[id] → Student details page

API Routes:
POST /api/auth/verify → Verify Google OAuth token
GET /api/auth/verify → Check authentication status
POST /api/auth/logout → Logout user
GET /api/search?name=query → Search students
GET /api/student/[id] → Get student details
